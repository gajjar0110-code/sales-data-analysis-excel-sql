# Sales Data Analysis Project

## Tools Used
- Excel (Pivot Tables, Charts, Slicers)
- SQL

## Project Overview
Analyzed sales data to identify revenue trends, top-performing products, and countries.

## Key Features
- Data cleaning and formatting
- Interactive Excel dashboard
- SQL queries for sales insights

## Insights
- Classic Cars generated highest revenue
- Sales peaked during specific months
- USA was the top-performing country
